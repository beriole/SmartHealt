Ce fichier contient les données exhaustives (JSON, Form-Data, paramètres) pour tester chaque route de l'API SmartHealth sur Postman.

============================================================
                        ⚠️ IMPORTANT ⚠️
- Pour toutes les routes protégées, ajoutez le header :
  `Authorization: Bearer <VOTRE_TOKEN_JWT>`
- Pour les requêtes d'upload (Avatar, Médicament, Pharmacie), utilisez le mode **form-data** dans le Body de Postman, et passez le type de la clé de "Text" à "File" pour uploader l'image.
============================================================

### 4. MEDICAMENTS (`/api/medicaments`) - (Réservé PHARMACIEN)

**GET /api/medicaments**
- Parcourir le catalogue global. Params : `?recherche=Doli&categorie=antalgique`

**POST /api/medicaments**
- Rôle Requis : `PHARMACIEN`
- Body type : **form-data**
- Clé texte : `nom_commercial` = Doliprane
- Clé texte : `dci` = Paracétamol
- Clé texte : `forme_galenique` = comprime
- Clé texte : `dosage` = 1000mg
- Clé texte : `categorie` = antalgique
- Clé texte : `est_generique` = false
- Clé texte : `necessite_ordonnance` = false
- Clé texte : `prix_indicatif_fcfa` = 1500
- Clé fichier : `image` = (Sélectionnez le fichier image de la boîte)

**PUT /api/medicaments/:id**
- Même principe en `form-data` pour la mise à jour par les pharmaciens.

---
### 5. PHARMACIES (`/api/pharmacies`)

**POST /api/pharmacies** (Réservé ADMIN)
- Body type : **form-data**
- Clé texte : `id_responsable` = UUID_DU_PHARMACIEN
- Clé texte : `nom_pharmacie` = Pharmacie du Grand Marché
- Clé texte : `numero_autorisation` = PHARMA-001
- Clé texte : `adresse` = Yaoundé, Centre-ville
- Clé texte : `latitude` = 3.8480
- Clé texte : `longitude` = 11.5021
- Clé texte : `telephone` = +237222000000
- Clé texte : `livraison_disponible` = true
- Clé texte : `rayon_livraison_km` = 15
- Clé fichier : `image` = (Photo de la façade de l'officine)

**PUT /api/pharmacies/:id** (Réservé ADMIN ou au PHARMACIEN responsable)
- Body type : **form-data** (Pour changer les horaires ou la vitrine)

**GET /api/pharmacies/:id_pharmacie/evaluate-ordonnance/:id_ordonnance**
- Permet de simuler le coût pour servir une ordonnance dans cette pharmacie.

---
### 6. GESTION DES STOCKS E-COMMERCE (`/api/stocks`) - (Réservé PHARMACIEN)

**GET /api/stocks/search**
- Accessible à tous. Cherche le produit dans les officines.
- Query : `?medicament=Doliprane&sort=price` ou `sort=rating`

**GET /api/stocks/my-stocks**
- Rôle requis : `PHARMACIEN` (Renvoie tout son propre inventaire).

**POST /api/stocks**
- Rôle requis : `PHARMACIEN` ou ADMIN
- Body type : `raw JSON`
{
  "id_medicament": "UUID_DU_MEDICAMENT",
  "quantite_disponible": 100,
  "prix_vente_fcfa": 2500,
  "seuil_alerte": 20,
  "date_peremption": "2025-12-31T00:00:00.000Z"
}

**PUT /api/stocks/:id_stock**
{
  "quantite_disponible": 85,
  "prix_vente_fcfa": 2400
}

**DELETE /api/stocks/:id_stock**
- Retire définitivement le médicament de la vente dans cette pharmacie.

### 9. CARNETS (`/api/carnets`)

**GET /api/carnets/my-carnet**
- Rôle : `PATIENT` (Renvoie son carnet)

**POST /api/carnets/my-carnet/regenerate-qr**
- Regénère la clé du carnet.

**GET /api/carnets/scan/:qrToken**
- Rôle : `MEDECIN`, `INFIRMIER`, `PHARMACIEN`. Scanner le QR Code du patient en consultation/pharmacie.
